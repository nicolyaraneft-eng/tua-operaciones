Notificaciones por correo

Cada vez que se crea un ticket, llega un correo automático a operacionestua20@gmail.com. Requiere configurar EmailJS (gratis).

Cómo configurar EmailJS:
1. Crea cuenta en emailjs.com
2. Agrega un Email Service (Gmail u Outlook) → copia el Service ID
3. Crea una plantilla con {{mensaje}} en el cuerpo y operacionestua20@gmail.com como destino → copia el Template ID
4. En Account → API Keys copia la Public Key
